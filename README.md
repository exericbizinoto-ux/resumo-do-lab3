# Resumo do Lab: Dominando Cloud e Boas Práticas de TI

Este repositório foi criado para consolidar os conhecimentos adquiridos durante o treinamento na DIO, focando em conceitos fundamentais de computação em nuvem, gestão de infraestrutura e documentação técnica.

## 🧠 Principais Aprendizados

### 1. Fundamentos de Cloud Computing
* **Modelos de Serviço:** Diferenciação entre IaaS, PaaS e SaaS, compreendendo a responsabilidade compartilhada entre o provedor e o cliente.
* **Escalabilidade e Disponibilidade:** Importância de arquiteturas resilientes e como a nuvem facilita o crescimento sob demanda.

### 2. Ecossistema Azure
* Exploração inicial dos recursos do **Microsoft Azure**, preparando a base para a certificação **AZ-900**.
* Conhecimento sobre regiões, zonas de disponibilidade e grupos de recursos para organização de ativos.

### 3. Versionamento com Git & GitHub
* Utilização do fluxo de trabalho básico: `init`, `add`, `commit` e `push`.
* A importância do arquivo **README.md** como a "vitrine" de um projeto técnico, facilitando o compartilhamento de conhecimento.

## 🛠️ Aplicação Prática
Durante o lab, pude conectar esses conceitos com minha experiência em administração de sistemas e suporte N2, visualizando como a transição para ambientes híbridos e cloud-native otimiza a entrega de serviços de TI.
